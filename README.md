# Material for MkDocs

This image is simply [`squidfunk/mkdocs-material`](https://github.com/squidfunk/mkdocs-material) + some plugins

For the list of added plugins refer: [requirements.txt](./requirements.txt)

Each image is tagged with the hash of the latest build commit, for example: `ghcr.io/dpi0/mkdocs-material:01e8aee`
